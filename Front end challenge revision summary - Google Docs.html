Front end challenge revision summary - 
 
 
For my revision request I first changed my HTML, replacing the many <div>s with unordered 
lists (<ul>) for each .bus_info_box class and lists (<li>) for each item inside. I also changed the 
.direction class items from <div>s to <section>s. Another step I took to make my code more 
semantic was renaming my classes from generic titles for everything to more item specific 
names that described the category of each item. 
 
For my visual revisions I changed my status bar to a more realistic version using a 
bold/semibold SF Pro Text font. I also changed my color palette to mostly a blue and tints 
thereof taken directly from Apples Human Interface Guidelines. My font-family was changed 
from Montserrat to SF Pro Text, which was not a huge change but still noticeable.
